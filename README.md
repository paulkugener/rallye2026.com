# Site Rallye 2026

Un site responsive simple construit avec **Picocss** et une ambiance musicale.

## 📦 Structure
- `index.html` - Accueil
- `information.html` - Informations & programme
- `teams.html` - Équipes et notation
- `sponsors.html` - Sponsors
- `signup.html` - Formulaire d'inscription
- `styles.css` - Ajustements de thème

## 🚀 Exécution locale
Ouvrez n'importe quel fichier `.html` dans un navigateur. Pour un meilleur résultat (et pour la soumission de formulaire plus tard), lancez un serveur local :

```bash
# Python 3
python -m http.server
```

Puis ouvrez `http://localhost:8000`.

## 🌍 Déploiement sur GitHub Pages
1. Créez un repo GitHub et poussez ce dossier.
2. Dans les paramètres du repo, activez GitHub Pages sur la branche `main` (ou `gh-pages`).
3. Votre site sera disponible à `https://<nom-utilisateur>.github.io/<repo>`.
